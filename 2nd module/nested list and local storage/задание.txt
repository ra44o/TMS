1. Создаём сервер
2. Создаём index.html
3. В index.html файле будет два элемента
3.1 Дерево типа https://learn.javascript.ru/task/sliding-tree
3.2 Таблица (используем bootstrap http://getbootstrap.com/docs/4.1/content/tables/)
4. При загрузке страницы загружаем json файл с иерархией компании (Приложение А)
5. Строим на основе него дерево (п. 3.1)
6. На каждый айдишник создаём на сервере файл Json с минумум одной записью сотрудника (имя, фамилия, возраст, id, номер), 
файл назвается типа id0.json, id1.json
7. При нажатии на элемент дерева, должен идти запрос на сервер и запрашивать файл с соответствующим id
(нажали на IT Department Head -- загрузился с сервера id0.json) -- Элемент дерева выделен, инфа о сотруднике появляется в таблице 
8. Вся инфа после загрузки хранится в LocalStorage под своим id (ключ == id), т.е сначала проверяем,
есть ли данные там, а потом делаем запрос на сервер если нужно
9. Есть кнопка "очистить" которая сбрасывает весь LocalStorage, выделение в дереве и таблицу

Можно использовать JQuery, можно чистый JS

Have fun coding!
